# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持自定义检测区域和报警设置
  - 目标类别选择
  - 声音报警开关
  - 目标图像记录保存
  - 滞留时间阈值设置
- 支持自定义置信度和IOU阈值
- 可选择是否显示标签和原图
- 提供多种检测方式：
  - 单张图片识别
  - 图片文件夹批量识别
  - 视频文件识别
  - 本地摄像头实时识别

## 适用场景
- 行人和车辆目标检测
- 区域入侵监测
- 视频监控分析
- 图像识别研究
- 计算机视觉教学

## 技术特点
- 基于YOLOv8目标检测算法
- 使用Python 3.9开发
- 集成OpenCV、PyQt5、Ultralytics等库
- 提供完整的UI界面
- 内置3000张VOC2007数据集（7类）
- 包含训练好的模型及训练结果
- 提供完整的项目运行教程
